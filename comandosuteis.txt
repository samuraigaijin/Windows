Comandos Úteis Windows - Segurança e Operação

tracert <ip/host> - Rastreia a rota dos pacotes até o destino (diagnóstico de rede)
pathping <ip/host> - Combina ping e tracert para identificar perda de pacotes em cada salto
nslookup <host> - Consulta servidores DNS para resolver nomes de domínio
getmac - Exibe os endereços físicos (MAC) de todas as placas de rede
powershell Get-FileHash -Algorithm SHA256 <arquivo> - Verifica a integridade (hash) de um ficheiro
cipher /w:C: - Sobrescreve dados eliminados para impedir recuperação forense
gpresult /r - Exibe o conjunto de políticas de grupo (GPO) aplicadas ao usuário/máquina
hostname - Exibe o nome da máquina
ver - Mostra a versão do sistema operacional
systeminfo - Relatório completo de hardware, hotfixes e SO
netstat -ano - Lista conexões de rede ativas e PIDs associados
ipconfig /all - Detalhes completos de todas as interfaces de rede
tasklist - Lista todos os processos em execução
whoami /all - Mostra o usuário atual e seus privilégios/grupos
net user - Lista os usuários locais
net localgroup administrators - Lista quem tem privilégios de admin local
qwinsta - Lista sessões de usuários logados (RDP e local)
dir /A /S - Procura arquivos ocultos em todos os subdiretórios
findstr /S /I "password" *.txt - Busca strings específicas em arquivos recursivamente
type arquivo.txt - Exibe o conteúdo de um arquivo no terminal
icacls "C:\caminho" - Verifica ou altera permissões de arquivos/pastas (sucessor do cacls)
wevtutil qe Security /f:text /c:10 /rd:true - Lê os últimos 10 eventos do log de Segurança
set - Exibe todas as variáveis de ambiente (útil para achar caminhos e domínios)
arp -a - Exibe a tabela ARP (descoberta de vizinhos na rede)
route print - Exibe a tabela de roteamento IP
net share - Lista todas as pastas compartilhadas na máquina
